ফেসবুক এর Back Up Code বের করার নিয়ম

  1. প্রথমে নিচের দেওয়া LINK এ ক্লিক করুন।
  2. Two Factor Authentication এ যান → Facebook select করুন → Code চাইলে code দিন → পাসওয়ার্ড দিয়ে নম্বর add করে Two Factor Authentication চালু করুন।
  3. এরপর Additional Method এ ক্লিক করুন → Recovery Code এ ক্লিক করুন → Code গুলো কপি করুন।
  4. Top up website এর Back Up Code ঘরে এই code গুলো দিন এবং শেষে / দিয়ে আপনার WhatsApp নাম্বার লিখুন।
